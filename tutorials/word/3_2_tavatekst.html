3.2 Tavatekst (Normal Text)


Vali laad Tavatekst (Normal text), parem hiireklõps laadil ja vali Muuda stiili (Modify).


Vorming (Format) → Font (Font)



Seadista järgnevalt:

  • Font (Font): Calibri;
  • Joondus (Align): Rööpselt (Align);
  • Suurus (Size): 12p;


Vorming (Format) → Lõigud (Paragraphs)

  • Reasamm (Line Spacing): 1,5.
  • Lõiguvahe enne (Paragraph spacing before): 6p;
  • Lõiguvahe pärast (Paragraph spacing after): 0p;
